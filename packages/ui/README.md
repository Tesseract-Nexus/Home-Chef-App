# UI Package

This package contains shared React components used throughout the HomeChef web application.

## Components

- `Button`: A simple, reusable button component.

## Usage

Components from this package can be imported directly into the `apps/web` application:

```tsx
import { Button } from '@homechef/ui';
```

This package is intended to grow with the application, housing common UI elements like form inputs, modals, cards, and more to ensure a consistent look and feel.
